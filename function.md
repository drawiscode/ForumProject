# Vue3 论坛项目 — 功能需求（逐步深入版）

## 一、基础核心功能（必做）
### 1. 用户体系功能
- 用户注册：支持账号 + 邮箱/手机号注册，表单校验、密码加密存储  （已实现）
- 用户登录：账号密码登录、登录状态持久化、退出登录  （已实现）
- 个人信息管理：查看/修改头像、昵称、个性签名，修改登录密码  （已实现）
- 找回密码

### 2. 帖子核心功能
- 帖子发布：支持标题、内容编辑，关联板块分类  
- 帖子管理：编辑、删除自己发布的帖子  
- 帖子展示：列表分页展示、详情页展示、浏览量统计  
- 帖子排序：按发布时间、热度排序  

### 3. 评论交互功能
- 评论发布：对帖子发表评论  
- 楼中楼回复：对评论进行二级回复  
- 评论管理：删除自己的评论、评论分页展示  

### 4. 基础互动功能
- 帖子点赞、评论点赞  
- 简单搜索：按帖子标题模糊搜索  

## 二、进阶体验功能（优化）
### 1. 板块分类系统
- 多级板块划分（如：技术交流、生活分享、问答求助）  
- 板块专属帖子列表、板块信息展示  

### 2. 用户成长体系
- 积分机制：发帖、评论、签到获得积分  
- 用户等级：积分对应等级，页面展示等级标识  

### 3. 内容体验优化
- 富文本编辑：支持文字格式、列表、换行等排版  
- 图片上传：帖子、评论配图支持  
- 帖子标识：置顶帖、精华帖等特殊标识  

### 4. 消息通知系统
- 评论/回复通知：他人回复自己时触发消息提醒  
- @提及通知：被用户@时接收消息  
- 系统公告通知：管理员发布全局公告推送  

### 5. 个人中心升级
- 我的发帖、我的评论、我的收藏列表  
- 消息中心：查看所有历史通知  
- 个人主页：对外展示用户信息与发布内容  

## 三、社区氛围功能（加分）
- 帖子收藏/取消收藏  
- 用户关注/取消关注  
- 每日签到功能  
- 热门帖子、最新帖子排行榜  
- 用户屏蔽/黑名单功能  

## 四、后台管理功能（必做）
### 1. 用户管理
- 查看全量用户信息、禁用/解禁违规用户  
- 权限分配：普通用户、版主、管理员三级权限  

### 2. 内容管理
- 帖子管理：查看、删除、置顶、加精/取消加精  
- 评论管理：删除违规评论、批量清理垃圾评论  

### 3. 板块管理
- 新增、编辑、删除板块，设置板块权限  

### 4. 系统管理
- 发布/编辑系统公告  
- 敏感词过滤配置  
- 网站基础信息配置（名称、LOGO）  

## 五、高级扩展功能（亮点）
- 全文搜索：支持帖子内容与标题的精准搜索  
- 实时消息：WebSocket 实现即时消息推送  
- 内容安全：XSS 过滤、敏感词自动屏蔽  
- 数据缓存：Redis 缓存热门帖子与用户信息  
- 响应式适配：兼容 PC 与移动端  
- 数据统计：用户量、发帖量、评论量可视化展示
